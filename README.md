# Doctor Appointment

This application allow patients to book an appointment with doctors based on their availability.

## Requirements

To know about the requirements go through the `book-an-appointment.md` file.

## Tech Stack

- Frontend
  - ReactJS
  - Node 18.17.1
- Backend
  - Spring Boot 3
  - Java 17
  - Maven 3.9.6
- Database
  - MySQL 8.0.36
